Robotic Arm

Project Description

This project incorporated all the knowledge that was learned in the Embedded Design course by linking hardware and software to control a robotic arm. We successfully were able to control the speed and angle placement of each of the 5 servos of the robotic arm through Simulink and C++.

Project Details

  • Used Simulink to generate PWM signals on the ZedBoard FPGA, which controlled servos

  • Created C++ files to connect the bluetooth of the Wiimote to interact with the FPGA
